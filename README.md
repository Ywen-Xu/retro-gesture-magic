## 功能

- 📷 实时摄像头滤镜（梦幻粉 / LOMO复古 / 忧鬱冰蓝 / 迪斯科金）
- ✋ **手勢互動**：掌心召唤爱心、双手交叉爆炸、单指碎钻跟随
- 🖌️ 像素涂鸦 + Y2K 贴纸拖拽
- 🎵 8-Bit 慢摇合成器
- 📸 自动拍照（静止检测 + 5连拍）
- 📱 手机/桌面自适应

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000`，授权摄像头即可使用。

## 技术栈

React 19 · TypeScript · Tailwind CSS 4 · Vite · MediaPipe Hands · Web Audio API
